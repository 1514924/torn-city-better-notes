# Better Notes for Torn City

A Chrome extension that adds a markdown editor to torn.com notes with live preview.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" in the top-right corner
3. Click "Load unpacked"
4. Select the `notes` folder from this repository
5. Visit torn.com and press `Cmd+[` (Mac) or `Ctrl+[` (Windows/Linux) to open the drawer

## Usage

- **Open/Close**: Press `Cmd+[` or `Ctrl+[` to toggle the notes drawer
- **Close**: Press `Esc` or click the Ã— button to close
- **Save**: Click "Save" or press `Cmd+S` / `Ctrl+S`
- **New Note**: Click "New" to create a new note
- **Switch Notes**: Use the dropdown to select different notes
- **Export**: Click "Export" to download the current note as a `.md` file
- **Delete**: Click "Delete" to remove the current note

## Features

- Split-pane editor with live markdown preview
- Syncs with your torn.com notes
- Export notes as `.md` files for backup or use in other editors
- Dark theme matching VS Code
- Keyboard shortcuts for quick access
- Support for basic markdown: headers, bold, italic, links, lists, code blocks, etc.
